# Tourify

Welcome to **Tourify**, a team project for building a modern and responsive travel & tourism website.

## Technologies

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* Git & GitHub

## Getting Started

### 1. Clone the repository

```bash
git clone <repository-url>
```

### 2. Open the project folder

```bash
cd tourify
```

### 3. Open the project

Open the project in **Visual Studio Code**.

### 4. Run the project

You can run the project by:

* Opening `index.html` directly in your browser, or
* Using the **Live Server** extension in VS Code (recommended).

## Team Workflow

* Create a new branch before starting your task.
* Work only on the issue assigned to you.
* Do **not** push directly to the `main` branch.
* Commit your changes with clear commit messages.
* Push your branch and create a Pull Request.

Example:

```bash
git checkout -b feature/contact-page
git add .
git commit -m "Create contact page"
git push origin feature/contact-page
```


## Team Rules

* Write clean and organized code.
* Use Bootstrap classes whenever possible.
* Keep the website responsive.
* Reuse components and styles.
* Test your page before creating a Pull Request.
* Do not modify another teammate's work unless discussed with the team.
